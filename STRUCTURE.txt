MODULAR ARCHITECTURE
====================

Independent Modules:

  Backend Module:
    - Standalone FastAPI service
    - Can run independently: python -m uvicorn backend.main:app
    - RESTful API design
    - No frontend dependencies

  Frontend Module:
    - Standalone React+Vite application
    - Can run independently: npm run dev
    - No backend build dependencies
    - Simple HTTP communication with API

  ML Module:
    - Encapsulated in backend/model_stub.py
    - Loads models from artifacts/
    - Separate from server logic
    - Can be packaged independently

  Configuration Module:
    - Centralized in backend/config.py
    - Environment-specific settings
    - Model paths, server ports, CORS origin

This modular design enables:
  - Independent deployment of backend/frontend
  - Easy testing of individual components
  - Clear separation of concerns
  - Easy maintenance and updates
  - CI/CD pipeline integration

RUNNING THE APPLICATION
=======================

Single Command (Recommended):
  python run_fullstack.py

This automatically:
  1. Creates .venv (if needed)
  2. Installs backend dependencies
  3. Installs frontend dependencies
  4. Starts backend on http://localhost:8000
  5. Starts frontend on http://localhost:4173
  6. Monitors both services

Alternative Options:
  python run_fullstack.py --backend-only        # Backend only
  python run_fullstack.py --frontend-only       # Frontend only
  python run_fullstack.py --skip-install        # Faster restart
  python run_fullstack.py --help                # Show options


FIRST TIME SETUP
