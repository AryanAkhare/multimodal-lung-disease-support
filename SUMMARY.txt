COMMAND TO RUN APPLICATION
==========================

PRIMARY COMMAND (Recommended):
  python run_fullstack.py

This single command:
  ✓ Creates .venv if needed
  ✓ Installs all backend dependencies
  ✓ Installs all frontend dependencies
  ✓ Starts FastAPI backend (http://localhost:8000)
  ✓ Starts React frontend (http://localhost:4173)
  ✓ Monitors both services
  ✓ Handles Ctrl+C gracefully

Alternative Options:
  python run_fullstack.py --backend-only       # Backend API only
  python run_fullstack.py --frontend-only      # Frontend UI only
  python run_fullstack.py --skip-install       # Faster startup (no reinstall)
  python run_fullstack.py --help               # Show all options
